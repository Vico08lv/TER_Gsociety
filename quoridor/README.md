## Déploiement 

Pour deployer le jeu en local :

Build le front Vue
```bash
  cd ./vue-client
```
```bash
  npm install
```
```bash
  npm run build
```

Lancer le serveur
```bash
  cd ../node-server
```
```bash
  npm install
```
```bash
  npm start
```

Accèder au jeu sur le localhost:3000
